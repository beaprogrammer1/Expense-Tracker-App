# Expense Tracker App

## 📱 Overview
An Android mobile application for tracking personal expenses with geolocation, charts, and offline storage.

## 🎯 Features
- User Registration & Login (Firebase Auth / SQLite)
- Add/Edit/Delete Expenses
- Location tagging with Google Maps
- Expense history with filters (date, category, location)
- Spending charts (Pie/Bar charts using MPAndroidChart)
- Light/Dark mode support
- Offline storage (Room Database)

## 🛠️ Technologies Used
- **Language**: Kotlin / Java
- **IDE**: Android Studio
- **Database**: Room Database (SQLite)
- **Authentication**: Firebase Auth
- **Maps**: Google Maps API
- **Charts**: MPAndroidChart
- **Design Patterns**: MVVM, Repository, Singleton

## 🔧 Setup Instructions (Detailed)
1. 1. Clone this repo:
   ```bash
   git clone https://github.com/YOUR_USERNAME/ExpenseTrackerApp.git
2. Open in Android Studio
3. Add your `google_maps_api.xml` in `app/src/debug/res/values/`
4. Add `google-services.json` in `app/` directory
5. Sync Gradle and Run
