## 📱 Project Overview

**Personal Expense Tracker** is a web-based application that helps users manage their daily expenses intelligently. Built with **Google Gemini AI**, this app not only records expenses but also provides smart insights and categorization using artificial intelligence.

This project was developed as part of **CMP7003 - Emerging Mobile Applications** assessment at Cardiff Metropolitan University.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI-Powered Categorization** | Automatically categorizes expenses using Gemini AI |
| 📊 **Interactive Dashboard** | Visual representation of spending patterns |
| 💾 **Local Storage** | All data saved in browser's local storage |
| 📱 **Responsive Design** | Works perfectly on mobile, tablet, and desktop |
| 🌓 **Dark/Light Mode** | Toggle between themes for comfort |
| 🔍 **Search & Filter** | Find expenses by date, category, or amount |
| 📈 **Monthly Reports** | Generate detailed spending reports |
| 🏷️ **Custom Categories** | Create and manage your own expense categories |
